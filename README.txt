MA Smart Roudlotul Jinan Gentasari - Static Website (Bootstrap + AOS)

Files included:
- index.html
- tentang.html
- program.html
- kontak.html
- style.css
- script.js

How to preview locally:
1. Extract the ZIP.
2. Open index.html in your browser (double-click). For full features (AOS, Bootstrap CDN), internet connection is required.
3. To host on GitHub Pages: push the files to a repository and enable Pages from the main branch (root).

Notes:
- Instagram iframe may be blocked by Instagram (X-Frame-Options). If it doesn't show, click the 'Buka Instagram' button to open the profile.
- YouTube playlist is embedded using the playlist embed URL.
- Dark mode preference is stored in localStorage.
